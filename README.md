# Griffonnage - Sync Server

> Scribbl careless drawings with friends, privately

Synchronize drawings from Griffonnage using [Socket-io](https://socket.io).

## Build Setup

```bash
$ npm install

$ npm run dev

$ npm run build
$ npm start
```
